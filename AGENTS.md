# Project instructions

- Inspect the repository, Git state, and relevant configuration before editing.
- Keep this repository limited to the `mahmadnet.github.io` user-site landing
  page and its repository documentation.
- Keep `index.html` and `styles.css` empty until landing-page implementation is
  explicitly approved.
- Do not copy content from `mahmadnet-website` or claim, redirect to, or
  configure `www.mahmad.net` for this repository.
- Do not add `CNAME`, deployment workflows, frameworks, package managers,
  dependencies, build tooling, JavaScript, analytics, tracking, cookies,
  external fonts, third-party assets, or unapproved project links.
- Preserve direct GitHub Pages publishing from `main` and the repository root.
- Validate all changes, run `git diff --check`, and present the complete diff
  for review before any commit-related action.
- Obtain explicit approval before staging, committing, pushing, or changing
  repository or GitHub Pages settings.
